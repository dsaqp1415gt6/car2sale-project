# Car2Sale

Car2Sale es un proyecto educativo desarrollado en curso "Diseño de Servicios y Aplicaciones" de la escuela de Castelldefels Ingeniería de Telecomunicación y Aeroespacial (EETAC). Car2Sale es un servicio web de venta de vehiculos de segunda mano construido a partir de una interfaz de software API que se gestiona a partir del diseño una pagina web y una aplicacion android. La web está desarrollado con Bootstrap, CSS y jQuery, mientras que la aplicacion Android  parte de un dispositivo con un sistema operativo (API 15 y superior).

### Requerimientos del sistema
Para instalar y ejecutar Car2Sale usted tendra de disponer de:

- JDK 7.0
- Maven
- Tomcat
- Apache
- MySQL

### Installacion

Sólo tienes que clonar este proyecto y siga las instrucciones de cada uno de los subproyectos individuales:

- sql
- www
- car2sale-servidor
- car2sale-android
